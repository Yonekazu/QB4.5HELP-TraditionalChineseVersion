STATIC - 為一 BASIC 的宣告敘述,使得一個簡單變數或陣列成為一個程序式
         (或一個 DEF FN 函數)的區域(local)變數,並且在不同的程序式呼叫
         期間仍保留其值。

語法
  STATIC 變數串列
    * 變數串列有下列的語法:
        變數[()] [AS 型態] [,變數[()] [AS 型態]] ...
      使用 "()" 來表示一個陣列。請參閱 <細節> 有更多有關 AS 型態的子句。

可參閱  <SUB>  <FUNCTION>  <DEF FN>
