WIDTH - 為一檔案輸入/輸出敘述,用來指定檔案或裝置(device)輸出列(Row)的
        寬度,或者改變螢幕上可以顯示的列(Row)和行(Column)的數目。

語法
  WIDTH [行數] [,列數]
  WIDTH {#檔案代碼 | 裝置}, 寬度
  WIDTH LPRINT 寬度

    * 請參閱<細節> 有說明 WIDTH 敘述參數的不同用法
    * 請參閱 <使用函數及敘述應注意的事項>內有說明 WIDTH LPRINT 寬度
      在中文版使用時應注意的事項。
