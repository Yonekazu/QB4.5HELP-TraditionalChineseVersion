PCOPY 敘述細節

語法
  PCOPY 原始分頁,目的分頁

原始分頁為一整數運算式,其值介於 0 到 n 之間,其中 n 是由目前的視訊記憶大小
(Video memory Size)和目前螢幕模式的每頁大小,來決定其最大的頁數。

目的分頁參數的需求與原始分頁參數相同。

可參閱 SCREEN 敘述有提供在不同螢幕模式下,有關的分頁資訊。
