SIN 函數程式範例

本範例繪製極座標方程式 r = 1 + sin n (θ)的圖形。這圖形為著名的 Cardioid 圖,
當 n 等於 1 時,會類似一個心臟線。

'*** SIN 函數的程式範例 ***
CLS
CONST PI = 3.141593
SCREEN 82 : COLOR 1              'VGA 繪圖模式,藍色的前景
WINDOW (-3,-2)-(3,2)             '轉換螢幕到直角座標
INPUT "花瓣的數目 = ", N
CLS
PSET (1,0)                       '設定起始點
FOR Angle = 0 TO 2*PI STEP .02
   R = 1 + SIN(N*Angle)          '"花"圖形的極座標方程式
   X = R * COS(Angle)            '轉換極座標到
   Y = R * SIN(Angle)            '直角座標
   LINE -(X,Y)                   '從上一點到此點繪製一直線
NEXT
END
