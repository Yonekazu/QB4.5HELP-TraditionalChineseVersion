REDIM 敘述程式範例

本範例顯示如何使用 REDIM 來配置記錄的陣列,然後如何釋放記錄所使用的記憶體。

TYPE KeyElement
   Word AS STRING * 20
   Count AS INTEGER
END TYPE

' 成為動態陣列
' $DYNAMIC
CLS                          ' 清除螢幕
' 當需要時,配置一個記錄陣列
REDIM Keywords(100) AS KeyElement
Keywords(99).Word = "ERASE"
Keywords(99).Count = 2
PRINT "關鍵字 99 為 "; Keywords(99).Word
PRINT "次數為"; Keywords(99).Count
' 當使用完成後,釋放關鍵字 Keywords 陣列所佔用的空間
ERASE Keywords
END

範例輸出

關鍵字 99 為 ERASE
次數為 2
