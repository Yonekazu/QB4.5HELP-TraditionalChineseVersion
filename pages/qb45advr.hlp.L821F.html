使用在檔案輸入/輸出的敘述和函數

任務                        敘述或函數             作用

產生一個新的檔案或存取      OPEN             開啟一個檔案以便讀取或寫入
一個已存在的檔案                             資料錄(Records)。

關閉一個檔案                CLOSE            結束一個檔案的輸入/輸出。

儲存資料到一個檔案內        PRINT #          儲存一連串的變數到一個先前
                                             已開啟的循序檔案中的資料錄
                                             欄位(Record Fields)內。

                            PRINT USING #    與 PRINT # 相似,但 PRINT
                                             USING # 會將資料錄的欄位
                                             格式化。

                            WRITE #          儲存一連串的變數到一個先前已
                                             開啟的循序檔中之資料錄欄位內。

                            WIDTH            指定一個標準長度給循序檔中
                                             的每一個資料錄。

                            PUT              儲存一個使用者定義的變數
                                             內容到一個先前已開啟的
                                             二進位檔或隨機存取檔中。

由檔案中擷取資料            CINPUT$          傳回由指定檔案所讀取的
                                             字元字串。

                            INPUT #          從一個循序檔中讀取一個
                                             記錄,並將記錄中的每一欄位
                                             指定給程式中的變數。

                            INPUT$           從一個檔案中讀取一組字元
                                             (以位元組為計算單位)字串。

                            LINE INPUT #     從一個循序檔中讀取一個
                                             記錄,並且把它存到一個單一
                                             字串變數中。

                            GET              從二進位或隨機存取檔中讀取
                                             資料,並將資料指定給一個
                                             使用者自定變數內的元素。

管理磁碟上的檔案            FILES            列出在指定目錄內的所有檔案。

                            FREEFILE         傳回下一個可用的檔案代碼。

                            KILL             由磁碟上刪除一個檔案。

                            NAME             改變一個檔案的名稱。

取得檔案訊息                EOF              測試是否所有的資料都已經從
                                             檔案中讀取完畢。

                            FILEATTR         傳回經由作業系統指定給一個
                                             開啟檔案的代碼,以及代表該
                                             檔案開啟時的存取模式(INPUT,
                                             OUTPUT,APPEND,BINARY,或
                                             RANDOM)。

                            LOC              傳回目前檔案指標的位置。
                                             在二進位存取模式中,它傳回
                                             位元組的位置。在循序存取
                                             模式中,則傳回位元組位置,
                                             並將此值除以 128。在隨機
                                             存取模式中,則傳回上次被
                                             讀/寫的記錄代碼。

                            LOF              傳回已開啟檔案的位元組數目。

                            SEEK (函數)      傳回下次將發生輸入/輸出操作
                                             的位置。在隨機存取模式中這是
                                             傳回下次將讀/寫記錄的記錄代碼。
                                             其它別種檔案存取,均傳回下次將
                                             讀/寫的位元組位置。

移動檔案裡的指標            SEEK (敘述)      設定在已開啟的檔案中,下次將被
                                             讀/寫的位元組位置。
