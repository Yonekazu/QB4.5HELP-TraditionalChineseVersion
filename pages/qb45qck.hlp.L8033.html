CONST - 為一不可執行敘述,用來宣告符號常數,以便取代程式中的數值或字串。

語法
  CONST 常數名稱 = 運算式 [,常數名稱 = 運算式]...
    * 常數名稱依據 BASIC 變數名稱的命名規則來命名。您可以在此名稱之後
      加上型態宣告字元(%, &, !, #, 或 $)來代表它的型態(這些字元不是
      名稱的一部份)。
    * 運算式是由一些文字,其他常數,或除了指數(^)以外的算術和邏輯運算子
      (Logical Operator)所組成。
