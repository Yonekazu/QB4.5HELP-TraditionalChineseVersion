INPUT - 為一裝置輸入/輸出敘述,用來在執行程式時,可以由鍵盤輸入資料
        ,並將其值存放到所指定的變數串列中。

語法
  INPUT[;] [" 提示字串 "{;|,}] 變數串列
    * 變數串列是一個或一個以上用逗號分開的變數。
    * 若有使用提示字串,會將此訊息先顯示在螢幕上,用來提示使用者現在
      要從鍵盤輸入什麼資料。
    * 字元 ";" 和 "," 是 <細節>。
    * 允許所輸入的字串資料可以包含中文字元與英文字元,而且允許使用者以
      "字元" 為單位,來移動游標或刪除,或插入資料。

可參閱  <INKEY$>

INPUT 關鍵字的其他使用法
  <LINE INPUT> - 從鍵盤讀取一整行的資料,並且將它存到一個字串變數中。
  <INPUT #> - 從循序檔中讀入資料項(或"欄位"),並且存到對應的變數串列中。
  <LINE INPUT #> - 從循序檔中讀取整筆記錄,並且存到一個字串變數中 。
