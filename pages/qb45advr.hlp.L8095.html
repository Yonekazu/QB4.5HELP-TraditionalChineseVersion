COS 函數程式範例

下列程式繪出極座標方程式 r = nθ 的圖形,其中 n 的值從 0.1 - 1.1 的圖形。
此程式中使用轉換因子 x = cos(θ) 和 y = sin(θ) 將極座標軸轉換為 x,y 直角
座標。此程式所畫出的圖形是著名的阿基米得螺旋(Spiral of Archimedes)。

CONST PI = 3.141593
SCREEN 71
'定義視窗大到可以容納最大的螺旋圖
WINDOW (-4,-6)-(8,2)
'從原點到右邊畫一直線
LINE (0,0)-(2.2*PI,0)
'繪製 10 個螺旋
FOR N = 1.1 TO .1 STEP -.1
   '繪出起始點
   PSET (0,0)
   FOR Angle = 0 TO 2*PI STEP .04
      '螺旋的極座標方程式
      R = N * Angle
      '轉換極座標到直角座標
      X = R * COS(Angle)
      Y = R * SIN(Angle)
      '由前一點到新點畫上一直線
      LINE -(X,Y)
   NEXT
NEXT
