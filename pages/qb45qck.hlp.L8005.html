  線上援助的使用方法如下:

    * 按下 F1 鍵 或觸動(click)滑鼠右鍵,可以隨時得到援助。
    * 在任何一個 BASIC 敘述,QuickBASIC 功能表項目或交談框上,
      按下 F1 鍵 時,就可以獲得援助。
    * 在援助視窗上可以隨時按下 ESC 鍵,將援助訊息由螢幕上移走;
      或者可以按下 PGDN 或 PGUP 鍵,來捲動螢幕上的援助訊息。

  所提供的援助資訊將顯示在 :
    * 螢幕頂端的援助視窗,或
    * 螢幕中間突出式的交談框。

  在援助視窗中也可以同時使用關係鏈來提供與此主題有關的其他援助,關係鏈包括:
    * 所有 BASIC 的關鍵字都是關係鏈。
    * 其他的關係鏈也可以被識別,因為 :
      - 都是用反白的角括弧包圍起來。
      - 而且,您可以按下 TAB 鍵來移動游標到一個關係鏈上。

  要顯示關係鏈上所參考的訊息,請將游標移到關係鏈上的任何位置,然後按下 F1 鍵
  或觸動滑鼠右鍵便可得到。

  秘訣:按 F6 或 SHIFT+F6 鍵 移動游標進入援助視窗,或移動滑鼠游標進入
       援助視窗並觸動左按鍵。

  QuickBASIC 會記錄最近 20 個所使用的關係鏈。您可以重覆 按 ALT+F1 鍵
  來回溯查閱。

  在援助視窗中,您也可以使用 QuickBASIC 功能表上的命令來 :
    * 複製本文(例如:使用 "C/複製" 命令來複製程式範例),然後離開援助視窗,
      再把它放到景象視窗中。
    * 在援助檔案中尋找您所需要的援助項目的其它用法。如果這個項目不是一個
      關係鏈,那將是非常有用的。

  F1 鍵和 ALT+F1 鍵兩者都是 "H/援助" 功能表命令的簡化鍵。按下 ALT+H 鍵
  拉下 "H/援助" 功能表,然後按下一個在 "H/援助" 功能表命令上符合您所要的
  反白英文字母鍵,來選取一個 "H/援助" 功能表中的命令。

  下面是 "H/援助" 功能表命令的摘要:

      命令           動作
    * 索引           在援助視窗中列出所有 BASIC 關鍵字的關係鏈。

    * 內容           在援助視窗中列出有關 BASIC 和 QuickBASIC 一套常用
                     的主題之關係鏈,例如: 顯示有關 BASIC 資料型態,或
                     運算式(expression)與運算子(operator)。

    * 主題:          顯示游標上所在項目的援助。

    * 線上援助       按下 SHIFT+F1 鍵 隨時顯示如何使用線上援助功能的
                     援助訊息。
