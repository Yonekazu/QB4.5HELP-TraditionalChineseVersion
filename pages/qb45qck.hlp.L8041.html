DEF型態 - 為一 BASIC 宣告敘述,用來設定變數, DEF FN 函數和 FUNCTION
          程序式的預設資料型態。

語法
  DEFINT 字母範圍 [,字母範圍] ...
  DEFSNG 字母範圍 [,字母範圍] ...
  DEFDBL 字母範圍 [,字母範圍] ...
  DEFLNG 字母範圍 [,字母範圍] ...
  DEFSTR 字母範圍 [,字母範圍] ...

    * 字母範圍有下列型式:
        字母 1[-字母 2]

      其中 字母1 和 字母2 大小寫均可使用,變數名稱和函數的名稱是以字母範圍
      中的字母為開始,其資料型態是由 DEF型態 敘述關鍵字中最後的三個英文字母
      (INT, SNG, DBL, LNG, 或 STR)來定義。
