ﾪﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬ 援助: 資料型態的關鍵字 ﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾬﾧﾡﾫﾬﾦ


在一個宣告敘述或參數串列中,用來對一個變數指定其型態(INTEGER, LONG, SINGLE,
DOUBLE, STRING, DOUBLE, STRING, 或者使用者自定型態)。

INTEGER  - 指定一個變數是佔 16 個位元(bits)長度的有正負號的整數。
LONG     - 指定一個變數是佔 32 個位元長度的有正負號的整數。
SINGLE   - 指定一個變數是單精度的浮點數。
DOUBLE   - 指定一個變數是倍精度的浮點數。
STRING   - 指定一個變數在 COMMON, DIM, REDIM, SHARED 或 STATIC 敘述中
           為字串。假如 STRING 有緊隨著 *n, 這字串的固定長度為 n 個位元組。
           否則,為變動長度的字串。

可參閱  <AS>  <COMMON>  <DECLARE>  <DEF FN>
          <DIM>  <FUNCTION>  <SHARED>
          <STATIC>  <SUB>  <TYPE>
