DEF FN - 為一程序式敘述,用來對一個函數做定義和命名。

單行語法
  DEF FN名稱[(參數串列)] = 運算式

區段語法
  DEF FN名稱[(參數串列)]
     [ 敘述區段 ]
  FN名稱 = 運算式
     [ 敘述區段 ]
  END DEF

  * FUNCTION...END FUNCTION 敘述提供另一種更直接的方法來對一個新的函數作
    定義和命名。如果必須使用 DEF FN 時,請參閱<細節> 有說明各種參數的意義。

可參閱  <FUNCTION> <STATIC>
