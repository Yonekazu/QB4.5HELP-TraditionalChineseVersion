LPOS 函數程式範例

此範例提示輸入球隊名稱及各球隊球員姓名,並將球隊及球員姓名送到印表機列印。


CLS                                ' 清螢幕
LPRINT"球隊成員"; TAB(76); "TEAM" : LPRINT
INPUT "有多少球隊"; TEAMS
INPUT "每隊有多少球員";PPT
PRINT
FOR T = 1 TO TEAMS
    INPUT "球隊名稱: ", TEAM$
    FOR P = 1 TO PPT
        INPUT " 輸入球員姓名: ", PLAYER$
        LPRINT PLAYER$;
        IF P < PPT THEN
            IF LPOS(0) > 55 THEN     '如果列表機表頭超過第 55 欄
                                     '則印出一空行
                LPRINT : LPRINT "     ";
            ELSE
                LPRINT ", ";         '否則印出一個逗號
            END IF
        END IF
    NEXT P
    LPRINT STRING$(80-LPOS(0)-LEN(TEAM$),"."); TEAM$
NEXT T
