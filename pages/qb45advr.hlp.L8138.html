ON...GOSUB 敘述程式範例

下列程式片斷根據 Chval 的值來決定控制權要分歧到 4 個所列的次常式中的那一個:

DO
    CLS
    PRINT "1) 顯示會議出席人數"
    PRINT "2) 計算註冊費用總額"
    PRINT "3) 列印郵寄清單"
    PRINT "4) 程式結束"
    PRINT : PRINT "請選擇?"
    DO
       Ch$=INKEY$
    LOOP WHILE Ch$=""
    Chval = VAL(Ch$)
    IF Chval > 0 AND Chval < 5 THEN
        ON Chval GOSUB Shop, Fees, Mailer, Progend
    END IF
LOOP
END
