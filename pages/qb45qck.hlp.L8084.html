OFF 關鍵字 - 不同的用法

  * OFF 關鍵字使用在不同的敘述上,來關閉下列事件的設陷:
      COM(n) OFF   - 發生在通訊埠(port) n 上的事件。
      KEY(n) OFF   - 發生在鍵值 n 上的按鍵事件。
      PEN OFF      - 光筆事件。
      PLAY OFF     - 發生在背景音樂緩衝區內的事件。
      STRIG(n) OFF - 發生在搖捍觸動 n 上的事件。
      TIMER OFF    - 時間事件。
      UEVENT OFF   - 使用者自定事件。

  * OFF 關鍵字也可使用來關閉在螢幕下端軟鍵(soft-key)的顯示:
      KEY OFF
