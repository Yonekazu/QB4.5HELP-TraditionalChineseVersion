SELECT CASE - 為一控制流程敘述,依據運算式計算的結果來執行數個敘述區段
              中的一個區段。

語法
  SELECT CASE 測試運算式
  CASE   運算式串列 1
    [ 敘述區段-1 ]
  [CASE   運算式串列 2
    [ 敘述區段-2 ]] ...

  [CASE ELSE
    [ 敘述區段-n ]]
  END SELECT

    * 測試運算式參數是任一數值或字串運算式。
    * 運算式串列參數包含一個或多個相同型態的運算式,其型態與測試運算式一樣。
      在一個運算式中的 IS 關鍵字,必須放在關係運算子(Relational Operator)的
      前面。
    * 敘述區段是由一行或多行以上的敘述所組成。
