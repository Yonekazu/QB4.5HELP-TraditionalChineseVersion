PEN 函數程式範例

下列範例產生無窮迴路來列印目前光筆開關的狀態(UP/DOWN)。按 CTRL+BREAK 則離開
迴圈。

             ------------- 注意 -------------
           若您的滑鼠驅動器仍開啟時,請勿執行此範例。
             --------------------------------

' 按下 CTRL+BREAK 鍵離開此迴圈
COLOR 0,7
CLS                ' 清螢幕
PEN ON
DO
   P = PEN(3)
   LOCATE 1,1 : PRINT "光筆為";
   IF P THEN PRINT "DOWN" ELSE PRINT "UP  "
   X = PEN(4) : Y = PEN(5)
   PRINT "X =" X, "Y =" Y; "  "
LOOP
COLOR 7,0
END
