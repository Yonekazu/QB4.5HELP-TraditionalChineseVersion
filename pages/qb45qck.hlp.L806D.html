LEFT$ - 為一字串處理函數,用來傳回從一個字串最左邊的 n 個位元組
        所組成的子字串。

語法
  LEFT$(字串運算式,n)
    * 字串運算式是一個字串常數,字串變數或字串運算式。
    * n 為一數值運算式(以 "位元組" 為計算單位),其值必需介於 0 和 32,767
      之間。
     假如 n 為 0,則傳回空字串(長度為 0 的字串)。

可參閱  <MID$ 函數> - 瞭解如何從一個字串的中間選取 n 個位元組。
           <RIGHT$> - 瞭解如何選取字串最右邊的 n 個位元組。
