IMP 邏輯運算子

語法
  結果 = 數值運算式 1  IMP  數值運算式 2

邏輯意含運算子(Logical-Implication operator), IMP 對數值運算式 1 和
數值運算式 2 相對應的位元(bit)做比較,然後依照下表,設定其值放在結果中:

───────────────────────────────────
  第一運算式中的位元        第二運算式中的位元          結果之位元
───────────────────────────────────
       1                         1                          1
       1                         0                          0
       0                         1                          1
       0                         0                          1
───────────────────────────────────
