OPEN - 為一檔案輸入/輸出敘述,用來開啟一個檔案或裝置的輸入/輸出。

語法
  OPEN 檔案名稱 [FOR 模式1] [ACCESS 存取] [鎖定] AS [#]檔案代碼 [LEN=記錄長度]
    * 檔案名稱是一個字串運算式,前面可以先有一個裝置(device)名稱,其後再跟著
      檔案名稱或路徑名稱。
    * 模式 1 可以為下列的幾種關鍵字之一:

RANDOM   讀取或寫入資料到隨機存取檔   | INPUT   僅能從循序檔(sequential file)
         (random-access file - 預設)  |         讀取資料
                                      | OUTPUT  寫入資料到循序檔
BINARY   讀取或寫入資料到二進位檔     | APPEND  寫入資料到循序檔的最末端
         (binary file)                |

    * 存取和鎖定 <細節> 僅在網路環境下使用。
    * 檔案代碼是一整數運算式,其值介於 1 和 255 之間。
    * 對隨機檔而言,預設的每筆記錄長度為 128 個位元組,而循序檔預設為 512 個
      位元組,若為二進位檔則可省略此參數。

可參閱  <FREEFILE>

OPEN 關鍵字的其他用法
  OPEN 模式 2, [#]檔案代碼, 檔名 [,記錄長度] - 為另一種語法(請參閱<細節>)。
  <OPEN COM> - 用來打開一個通訊通道(communication's channel)。
