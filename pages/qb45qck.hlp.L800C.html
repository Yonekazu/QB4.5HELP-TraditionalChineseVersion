ACCESS 子句

語法
  {ACCESS READ | ACCESS WRITE | ACCESS READ WRITE}

ACCESS 子句使用在 OPEN 敘述中,它只能在具有支援網路系統的 DOS 版本上
使用(如 DOS 3.0 版本以上)。

  * READ          打開一個只能讀取的檔案。
  * WRITE         打開一個只能寫入的檔案。
  * READ WRITE    打開一個能讀取且能寫入的檔案,此種模式僅在 RANDOM 和
                  BINARY 檔案,和使用 APPEND 的檔案中才有效。

可參閱  <OPEN>
