STRING$ 函數程式範例

範例 1

下例使用 STRIG$ 來產生一個報表頭:

Dash$ = STRING$(10,45)
PRINT Dash$;"月報表";Dash$

範例輸出

----------月報表----------


範例 2

下例程式使用 STRING$ 來產生一長條圖:

PRINT TAB(7);"西雅圖的日平均溫度"
PRINT
'取得各月份的資料及圖形
FOR Month = 1 TO 12 STEP 2
    READ Month$, Temp
    '列印(Temp-35)個星號
    PRINT Month$;" +"; STRING$(Temp-35,"*")
    PRINT "    |"
NEXT Month

'列印水平線
PRINT "    +";
FOR X = 1 TO 7
    PRINT "----+";
NEXT X
PRINT

'列印溫度標記
FOR X = 4 TO 39 STEP 5
    PRINT TAB(X); X+31;
NEXT X
PRINT

DATA Jan, 40, Mar, 46, May, 56
DATA Jul, 66, Sep, 61, Nov, 46
