POS 函數程式範例

本範例使用 POS 函數在每 40 個英文字元之後換到新的一行。

CLS                       ' 清螢幕
PRINT "本程式在每次鍵入 40 個英文字元之後會換行"
PRINT "請鍵入字元,按 <CTRL+C> 時,則結束。"
PRINT
DO
   DO WHILE POS(0) < 41   '停留在同一行,直到輸入 40 個英文字元為止
      DO
         Char$ = INKEY$
      LOOP WHILE Char$ = ""
      '如果輸入的組合鍵是 <CTRL+C> 則結束;否則印列字元
      IF ASC(Char$) = 3 THEN END ELSE PRINT Char$;
   LOOP
   PRINT                  '換到新的一行
LOOP
