DRAW - 為一繪圖敘述,用來畫出由字串運算式所定義的物體。

語法
  DRAW 字串運算式
    * 字串運算式是一個或一個以上繪圖命令所組合而成。

                                繪圖命令
──────── 用來畫線,或移動目前繪圖游標,或前兩者均動作 ────────
B   選項      - 移動游標, 但不畫線   │ N   選項      -  畫線後游標回原處
Un  向上           Dn  向下          │ Ln  向左             Rn  向右
En  向右上方       Fn  向右下方      │ Gn  向左下方         Hn  向左上方
Mx,y  移動到螢幕的 x,y 點座標上 (若 x 之前加上 + 或 - 時,所移動的是相對座標)

───────────────── 使用顏色  ────────────────
Cn  改變繪圖(前景)顏色               │ Pp,b  使用顏色 p 塗滿由顏色 b 所圍成
    (由 SCREEN 模式來決定 n )        │       的封閉圖形。

─────────────────使用比率和旋轉───────────────
Sn  增加或減少移動的單位長度         │ An   旋轉 (n * 90) 角度, 其中
    (n = 4, 是預設值)                │      n = 0, 1, 2, 或 3 。
    移動的單位長度 = n/4             │ TAn  旋轉 (-360 <= n <= 360) 度
    利用 Sn 可將圖形作放大或縮小     │

────────────────用來執行子字串────────────────
     X + VARPTR$(字串運算式)         │ 執行其它命令字串

可參閱  <SCREEN>  <PALETTE>  <VARPTR$>
