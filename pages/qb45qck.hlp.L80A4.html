RUN - 為一控制流程敘述,用來重新啟動目前存在主記憶體中的程式,或執行一個
      指定的程式檔。

語法
  RUN [{ 行號 | 檔名 }]
    * 若參數省略,則由目前載入程式的主模組中第一個可執行敘述開始執行。
    * 行號參數明確地指定主模組開始執行的起始行號。
    * 檔名是一字串運算式,將所指名的程式檔載入到主記憶體,並且執行。
