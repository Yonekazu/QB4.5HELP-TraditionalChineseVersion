FUNCTION...END FUNCTION - 為一不可執行敘述,用來宣告一個 FUNCTION 程序式,
                          參數串列和構成 FUNCTION 主體的程式碼。

語法
  FUNCTION 名稱[(參數串列)] [STATIC]
      [ 敘述區段 ]
    名稱 = 運算式
      [ 敘述區段 ]
  END FUNCTION

    * 名稱,根據與 BASIC 變數命名相同的原則來決定函數傳回的資料型態。
    * (參數串列)是用逗號來分開的一個或一個以上的變數,當 FUNCTION 程序式
      被呼叫時,這些參數將會傳給該 FUNCTION 程序式。
    * 運算式是 FUNCTION 所傳回來的值。
    * 關鍵字 STATIC 的使用,請參閱 <細節>。

可參閱  <DECLARE>  <STATIC 敘述>  <DEF FN>  <SUB>
