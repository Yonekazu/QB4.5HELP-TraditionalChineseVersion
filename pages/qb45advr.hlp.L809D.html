CDBL 函數程式範例

下列範例說明如何使用 CDBL 函數將數值運算式轉換成倍精確度數:

X = 7/9
X# = 7/9
PRINT X
'X# 和 CDBL(X) 的正確性僅小數點後的 7 位數,因為 7/9 是以單精確度來計算的。
PRINT X#
PRINT CDBL(X)
'精確性到小數點後的 15 位。
PRINT 7#/9#

範 例 輸 出

 .7777778
 .7777777910232544
 .7777777910232544
 .7777777777777778
