INPUT 敘述程式範例

此範例使用INPUT 敘述來輸入一個圓的半徑,並計算其面積。
此程式提示您輸入一個新的半徑並執行,直到輸入 0 才停止執行。

CLS
PI = 3.141593 : R = -1
DO WHILE R
    PRINT "輸入半徑 (0 表示離開)"
    INPUT ; "若半徑 = ", R
    IF R > 0 THEN
        A = PI * R ^ 2
        PRINT ",則圓面積 ="; A
    END IF
    PRINT
LOOP

範例輸出

輸入半徑 (0 表示離開)
若半徑 = 3,則圓面積 = 28.27434

輸入半徑 (0 表示離開)
若半徑 = 4,則圓面積 = 50.26549

輸入半徑 (0 表示離開 )
若半徑 = 0
